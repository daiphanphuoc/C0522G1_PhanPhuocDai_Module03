[Bài đọc] Nhúng CSS vào trong tài liệu HTML

Cách tiếp theo để khai báo các thuộc tính CSS đó là sử dụng thẻ style trong một tài liệu 'HTML'. Bên trong thẻ style chúng ta có thể đưa ra tất cả các khai báo CSS cần thiết rồi sau đó áp dụng cho từng thẻ 'HTML'. Thông thường, thẻ style được đặt bên trong thẻ head, nhưng đó không phải là bắt buộc, chúng ta có thể đặt chúng ở bất cứ đâu. Một tài liệu 'HTML' có thể có rất nhiều thẻ style khác nhau.
